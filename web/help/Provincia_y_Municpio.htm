  1. El botón Adicionar  en el panel Provincia permite adicionar una provincia dando la posibilidad al usuario de especificar los datos necesarios para su registro como el Nombre y la especificación del Código. Al hacer clic en el mismo se muestra el siguiente formulario  que permite insertar los datos especificados por el usuario.  

 

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones .

  • Debe tener en cuenta que no deben existir provincias con el mismo número de código.

 

  1. Para habilitar el botón Modificar del panel Provincias previamente se debe seleccionar  la provincia que se desea modificar, al seleccionar la opción se muestra el formulario Modificar Provincia con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

 

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

  •  Al seleccionar la provincia se activa el panel Municipio con los municipios asignados al elemento seleccionado.

 

  1. Para habilitar el botón Eliminar   del panel Provincia previamente se debe seleccionar la provincia que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

 

  •  Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .

  • Al seleccionar la provincia se activa el panel Municipio con los municipios asignados al elemento seleccionado.

 

  1. El botón Adicionar  en el panel Municipio permite adicionar un municipio a la provincia seleccionada dando la posibilidad al usuario de especificar los datos necesarios para su registro como el Nombre. Al hacer clic en el mismo se muestra el siguiente formulario  que permite insertar los datos especificados por el usuario.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Modificar del panel Municipio previamente se debe seleccionar  la provincia acto seguido el municipio que se desea modificar, al seleccionar la opción se muestra el formulario Modificar Municipio con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Eliminar del panel Municipio previamente se debe seleccionar  la provincia acto seguido el municipio que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.
