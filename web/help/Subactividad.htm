Subactividad

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Subactividad se muestra la interfaz Subactividades la cual permite gestionar las subactividades agrupadas dentro de las actividades mediante las opciones Adicionar, Modificar y Eliminar. Las subactividades existentes en el sistema se listan mediante el elemento Nombre. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación de la subactividad que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

  1. Para adicionar una subactividad se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir la denominación de la subactividad que se desea adicionar así como a la actividad que pertenece en el sistema. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la subactividad que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar subactividad con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la subactividad que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
