Entrega de Chips

  • Al seleccionar la opción del menú Economía / Entrega de Chips se muestra la interfaz Chips listándose la entrega de los chips definidos en el sistema agrupados por tarjetas mediante los datos Fecha de Servicio, Fecha de Entrega, Tipo Combustible, No. del Chip, Matrícula y Nombre del Chofer. En la interfaz se muestran las opciones que permiten  gestionar la entrega de los chips mediante las opciones: Adicionar, Modificar y Eliminar y la obtención de datos mediante la opción Imprimir.  La interfaz muestra la opción de realizar una búsqueda avanzada introduciendo el número de tarjeta, el número de vale y la fecha del período que desea buscar, posteriormente se ejecuta la búsqueda y se realiza la limpieza de la lista mediante los botones .  Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

  1. Para adicionar una entrega de chip antes se debe seleccionar la unidad a la que pertenece, se habilita la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar previamente se debe seleccionar la entrega de chip que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar Chip con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la entrega de chip que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Al seleccionar el botón Imprimir se muestra  la interfaz Tarjeta donde se debe seleccionar la tarjeta que desea, mostrando el reporte de la entrega de chips de dicha tarjeta,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada.

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.
