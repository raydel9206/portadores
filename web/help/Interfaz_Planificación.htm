Planificación

  • Al seleccionar la opción del menú Portadores / Combustible / Planificación se muestra la interfaz  Plan de Combustible listándose los planes definidos en el sistema agrupados por tipo de vehículo. Los datos que se muestran en el listado son editables lo que da una posibilidad al usuario de replanificar por meses a su necesidad de este portador. En la interfaz se muestran las opciones que permiten gestionar los planes de combustible mediante las opciones Guardar, Aprobar y  Desaprobar, y la obtención de datos Imprimir  y Exportar. La interfaz muestra la opción de realizar una búsqueda introduciendo la matrícula del vehículo y/o el tipo de combustible que se desea encontrar, además la opción de seleccionar el año y el mes que desea buscar. Al realizar una búsqueda anual se muestran los vehículos activándose los datos que especifican los Totales, Acumulados Mensuales ( Comb(L), Kms, Lubricante y Líquido de Freno) y Aprobado. Si realiza una búsqueda especificando el mes se muestran los vehículos activándose los datos que especifican los Acumulados Mensuales y el del propio mes ( Comb(L), Kms, Lubricante y Líquido de Freno) y Aprobado. Esta actividad pasa por un proceso de Aprobación luego que se establezca como aprobado un plan mensual el sistema no permite que los mismos se modifiquen.

 

 

  1. El botón Menú posee la opción de Guardar los datos, esta se habilita al dar doble clic encima del valor que se quiere modificar, la opción Aprobar permite aprobar el Plan de combustible seleccionado, se habilita cuando existe un plan sin aprobar, el mismo no podrá ser modificado ni eliminado, la opción Desaprobar permite desaprobar el Plan de combustible una vez que haya sido aprobado.

 

 

  1. El botón Imprimir posee la opción de Imprimir la planificación de combustible una vez que hayan sido aprobada,  así como la opción del navegador ( Google Chrome) de confirmación para realizar la función señalada.

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar. 

 

  1. Si se selecciona el botón Exportar, se muestra  la interfaz Exportar accidentes donde le permite introducir los datos necesarios, posteriormente se muestra la opción del navegador  con la cual se obtiene el documento Actas de responsabilidad Material en formato .doc. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.
