CDA002

 

  • Al seleccionar la opción del menú Reportes / CDA001 se muestra la interfaz  de igual nombre mostrando los datos en la plantilla obtenida con los resultados del proceso de gestión del Sistema Portadores Energéticos. La interfaz muestra  las opciones que permiten generar el reporte CDA002 y guardarlo mediante las opciones: Generar CDA001 y Guardar CDA001, así como obtener el registro de los datos mediante las opciones Imprimir y Exportar. En la interfaz se muestra la opción de realizar una búsqueda seleccionando el tipo de portador y la fecha, la cual evidencia el consumo real en el período seleccionado, teniendo en cuenta la gestión de los datos asociados al portador.

 

 

  1. Para habilitar la opción Generar CDA002 previamente se debe seleccionar la unidad y el portador a la que pertenece, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Para habilitar la opción Guardar CDA002 previamente se debe dar doble clic en los datos, dando la posibilidad al usuario de actualizar los mismos para su registro, posteriormente al hacer clic en el botón Guardar CDA002 se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Al seleccionar el botón Imprimir se muestra el reporte de el Modelo CDA002,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada.

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.

 

  1. Al seleccionar el botón Exportar  se muestra la opción del navegador que permite obtener la información en un documento excel (.xls).
