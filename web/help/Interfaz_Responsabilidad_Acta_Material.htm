Responsabilidad Acta Material

 

  • Al seleccionar la opción del menú Datos Primarios / Economía / Responsabilidad Acta Material se muestra la interfaz Responsabilidades del Acta Material la cual permite gestionar el concepto de responsabilidad del acta material mediante las opciones Adicionar, Modificar y Eliminar. Las responsabilidades del acta material existentes en el sistema se listan mediante el elemento Nombre. La interfaz muestra la opción de realizar una búsqueda introduciendo el nombre de la responsabilidad del acta material que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

  1. El botón Adicionar permite registrar una breve descripción de la responsabilidad, especificando con datos claros y concisos la intensión del concepto responsabilidad que se desea adicionar, al hacer clic en el mismo se muestra el siguiente formulario, que permite introducir el dato necesario para su registro. Los campos señalados indican que no pueden quedar vacíos para ser adicionado correctamente.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la responsabilidad que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar responsabilidad con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la responsabilidad que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones
