Unidad

 

  • Al seleccionar la opción del menú Datos Primarios / Estructura / Unidad se muestra la interfaz Unidades listándose las unidades existentes en el sistema mediante los elementos Siglas, Código, Municipio, Provincia  y Tipo. Se mostraran los elementos definidos en dependencia del nivel estructural que se le especifique a la Unidad. Se muestran las opciones que permiten gestionar las unidades  como: Adicionar, Modificar y Eliminar.

  • Para definir una Unidad debe conocer que: Uno de los eslabones básicos de dicho Sistema, lo constituye la Unidad Empresarial de Base (UEB), entendida como ese segmento o división interna que puede ser creada por la empresa y la Organización Superior de Dirección Empresarial (Osde), en consonancia con el Decreto-Ley 252 del 7 de agosto del 2007, en versión actualizada, revisada y concordada en fecha más reciente.

 

 

 

  1.  El botón Adicionar permite adicionar una Unidad dando la posibilidad al usuario de especificar los datos necesarios para su registro como el Nombre,  las Siglas, el Código, la condicionante Empresa Mixta, la especificación del Nivel estructural (OSDE, EMPRESA o UEB), y la precedencia definida por la Provincia y el Municipio. Al hacer clic en el mismo se muestra el siguiente formulario  que permite insertar los datos especificados por el usuario.  

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar  la unidad que se desea modificar, al seleccionar la opción se muestra el formulario Modificar Unidad con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

  • Debe tener en cuenta que si modifica la estructura de una Unidad esto requerirá cambios en el sistema y su funcionamiento.

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la unidad  que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .

  • Debe tener en cuenta que si elimina la estructura de una Unidad esto requerirá cambios en el sistema y su funcionamiento.
