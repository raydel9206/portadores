Cierre Mensual

 

  • Al seleccionar la opción del menú Transporte / Reportes / Cierre Mensual  se muestra la interfaz Conciliación Combustible Cierre Mes mostrando los datos en la plantilla obtenida con los resultados del proceso de gestión del Sistema Portadores Energéticos agrupados por actividad además en la parte inferior mostrando un resumen de estos datos. La Interfaz muestra las opciones de obtención de datos Imprimir y Exportar. Se muestra la opción de realizar una búsqueda mediante la selección  de la fecha, listándose automáticamente los elementos encontrados.

 

  1. Al seleccionar el botón Imprimir se muestra el reporte de el análisis de los consumos equipo a equipo,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada.

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.

 

  1. Al seleccionar el botón Exportar  se muestra la opción del navegador que permite obtener la información en un documento excel (.xls).
