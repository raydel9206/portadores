Descripción del Trabajo con la Aplicación

  • Luego de cumplir con los requisitos básicos se accede al sistema mediante el navegador determinado en este caso se utiliza el  Mozilla Firefox donde se especifica la siguiente dirección ejemplo: https://sigportadoresenergeticos.cu

 

La dirección que se especifica en el navegador dependerá de la ubicación del sistema.

 

 

  • Posteriormente se muestra la interfaz de acceso del sistema SIG-Portadores Energéticos en la cual se podrán observar los campos a introducir para acceder a las funcionalidades del sistema. El sistema está diseñado en un ambiente amigable contribuyendo al requisito no funcional de usabilidad. Mostrando entre una de sus opciones la posibilidad de visualizar y consultar una breve información sobre el objetivo y función del Sistema SIG-Portadores Energéticos así como contiene un menú de acceso rápido que posibilita al usuario gestionar los conceptos relacionados con el flujo del proceso de portadores.

 

        1. Se introduce el nombre del usuario que iniciará la sesión.

        2. Se introduce la contraseña perteneciente al usuario que iniciará la sesión.

        3. Permite verificar los datos introducidos con anterioridad y acceder al Sistema.

 

  • La aplicación está protegida por una Administración que es la encargada según el nivel del usuario; de mostrar las funcionalidades que éste pueda gestionar. La primera interfaz que se muestra al usuario es la presentación del SIG Portadores Energéticos, donde se introduce el usuario y la contraseña, si estos datos son válidos podrá tener acceso al Sistema y a sus funcionalidades tocando la tecla Enter o seleccionando el botón Iniciar sesión .  La aplicación cuenta con 3 Subsistemas (Administración, Portadores Energéticos y Mapa) dividiendo las funcionalidades por clasificación en cada una de ellas, los usuarios podrán tener acceso a las mismas en dependencia del nivel de acceso que estas requieran. El subsistema Administración se encarga del control de los datos y del acceso de las funcionalidades. El subsistema Portadores Energéticos es el encargado de llevar la gestión diariamente de la información de los portadores energéticos, y las producciones, permitiendo obtener resultados en un menor tiempo. El subsistema Mapa ofrece la posibilidad de visualizar los puntos de interés y las rutas creadas por las necesidades del flujo de trabajo.
