Distribución

  • Al seleccionar la opción del menú Portadores / Combustible / Distribución se muestra la interfaz Distribución de Combustible listándose las distribuciones existentes en el sistema agrupadas por tipo de combustible mediante los elementos Denominación, Fecha, Cant(L) y Aprob. Se muestra la opción de realizar una búsqueda introduciendo la fecha que desea buscar, además se muestran las opciones que permiten gestionar los datos de las distribuciones como: Gestionar( Adicionar, Modificar, Eliminar), Aprobar, Desaprobar e Imprimir. Al seleccionar una distribución se habilita a la derecha de la interfaz un panel denominado Desglose de Combustible listándose mediante los elementos Matrícula, Asign(L), Asign($), Km/L, Kms, Incremento(L) y Reducción(L) permitiendo gestionar mediante las opciones Actualizar, Deshacer y Reiniciar, se muestra la opción de realizar una búsqueda por matrícula del vehículo empleado que se desea encontrar, además se muestra la disponibilidad con que cuenta el tipo de combustible seleccionado.

 

 

  1. El botón Menú posee la opción de Gestionar los datos, para habilitar la opción del menú Gestionar previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así adicionar, modificar y eliminar una distribución, dando la posibilidad al usuario de especificar los datos necesarios para su registro.

  1.1 Para habilitar el botón Adicionar  previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así adicionar una distribución dando la posibilidad al usuario de especificar los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  •  Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1.2 Para habilitar el botón Modificar  previamente se debe seleccionar la distribución que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar distribución con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones   

 

  1.3 Para habilitar el botón Eliminar  previamente se debe seleccionar la distribución que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. El botón Menú posee la opción de Aprobar las distribuciones de combustibles, se habilita cuando existe una distribución sin aprobar, la misma no podrá ser modificada ni eliminada sin antes desaprobarla.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. El botón Menú posee la opción de Desaprobar las distribuciones de combustible una vez que hayan sido aprobadas.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. El botón Menú posee la opción de Imprimir las distribuciones de combustible una vez que hayan sido aprobadas y listas para su uso,  así como la opción del navegador ( Google Chrome) de confirmación para realizar la función señalada. En el documento en la parte superior izquierda se muestra el logo de la Entidad correspondiente.

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar. 

 

  1. Para habilitar el botón Actualizar y Deshacer previamente se debe dar doble clic en la asignación de combustible que desea modificar, dando la posibilidad al usuario de actualizar y deshacer los datos necesarios para su registro. Al hacer clic en el botón Reiniciar se reinician las operaciones realizadas en el desglose del combustible y se muestra el siguiente formulario.
