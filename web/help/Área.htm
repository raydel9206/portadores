Área

 

  • Al seleccionar la opción del menú Datos Primarios / Estructura / Área se muestra la interfaz Áreas la cual permite gestionar las áreas existentes en el sistema mediante las opciones Adicionar, Modificar  y Eliminar. Se mostraran los elementos definidos en dependencia del nivel estructural que se le especifique a la Unidad. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación del área que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

  1. Para habilitar el botón Adicionar  previamente se debe seleccionar la unidad a la que pertenece, permitiendo así adicionar un área dando la posibilidad al usuario de especificar los datos necesarios para su registro como el Nombre. Al hacer clic en el mismo se muestra el siguiente formulario  que permite insertar los datos especificados por el usuario.  

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar el área que se desea modificar, al seleccionar la opción se muestra la interfaz Modificar Área. Este formulario solo modifica la denominación del área.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

  • Debe tener en cuenta que si modifica la estructura de un Área esto requerirá cambios en el sistema y su funcionamiento.

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el área  que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .

  • Debe tener en cuenta que si elimina la estructura de un Área esto requerirá cambios en el sistema y su funcionamiento.
