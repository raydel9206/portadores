Requisitos Básicos para el Funcionamiento del Sistema

  • Los requerimientos de hardware son las características que debe tener el hardware de una computadora para poder soportar y/o ejecutar una aplicación o soportar un dispositivo específico. Usualmente se emplea para saber si una aplicación podrá funcionar correctamente en una determinada configuración de hardware de computadora. Debe tener en cuenta que los requerimientos de hardware y los requerimientos de software forman en su conjunto los requerimientos del sistema.

 

 

 Software (SIG Portadores Energéticos)

  • Los usuarios que explotarán este sistema deben tener conocimientos básicos del Sistema Operativo Windows y del flujo del proceso de gestión de Portadores Energéticos.

Hardware

  • Computadora 2da Generación (Dual Core) o superior.

  • Memoria RAM  4GB  o superior.

  • Disco Duro 120 GB o superior.

  • Tarjeta de Red de 1GB/s.

  • Sistema Operativo Windows o Linux.

  • PostgreSQL 9.4 ó superior.

  • PostGis 2.0. ó superior.

  • Mozilla Firefox 48.0 ó superior.

  • Google Chrome Versión 66.0.3359.139 ó superior.
