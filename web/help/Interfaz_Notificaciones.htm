Notificaciones

  • Al seleccionar la opción del menú Notificaciones se muestra la interfaz de igual nombre la cual permite gestionar las notificaciones que se definen en el sistema especificando el estado de las mismas listándolas por los elementos: Mensaje, Tipo, Usuario, Creado en y Leído en. El sistema muestra las opciones de gestión  Adicionar una notificación, así como Modificar y Eliminar una notificación previamente seleccionada. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1. Al hacer clic en la opción Adicionar se muestra el siguiente formulario que permite introducir la  especificación de la notificación así como definir el alcance que tendrá  la misma. Los campos señalados indican que son obligatorios y no deben quedar vacío.

 

  • Para guardar los datos o cancelar la operación se muestran al final de la interfaz las opciones .

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la notificación que se desea modificar, al seleccionar la opción Modificar se muestra  la interfaz Modificar notificación. Este formulario solo modifica la especificación de la notificación.

 

 

  • Se modifican los datos deseados,  para actualizar los cambios realizados de los datos o cancelar la operación se muestra al final de la interfaz las opciones . 

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la notificación que se desea eliminar, luego se muestra el mensaje de Confirmación.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de Confirmación las opciones  .
