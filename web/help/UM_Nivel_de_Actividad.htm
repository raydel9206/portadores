UM Nivel de Actividad

 

  • Al seleccionar la opción del menú Datos Primarios / Gestión / UM Nivel de Actividad se muestra la interfaz UMs del nivel de actividad la cual permite gestionar las unidades de medida de los niveles de actividad mediante las opciones Adicionar, Modificar y Eliminar. Las unidades de medida de los niveles de actividad existentes en el sistema se listan mediante el elemento Nombre. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación de la unidad de medida de la actividad que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

 

  1. Para adicionar una unidad de medida de los niveles de actividad se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir la denominación de la unidad de medida que se desea adicionar. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la unidad de medida de los niveles de actividad que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar UM del nivel de actividad. Este formulario solo modifica la denominación de la unidad de medida.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la unidad de medida de los niveles de actividad que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
