Cheque FINCIMEX

  • Al seleccionar la opción del menú Economía / Cheque FINCIMEX se muestra la interfaz Gestión de cheques para pago a FINCIMEX listándose los cheques definidos en el sistema en cada unidad mediante los datos No. Cheque, Moneda, Montos (Gasolina, Diesel y Total), Fecha de Registro, Depositado y Fecha de Depósito. Permite gestionar el concepto cheque mediante las opciones Adicionar, Modificar, Eliminar y Depositar y la obtención de datos mediante las opciones Comprobante, Imprimir y Saldos. La interfaz muestra la opción de realizar una búsqueda introduciendo el número del cheque que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

  1. Para adicionar un cheque antes se debe seleccionar la unidad a la que pertenece, se habilita la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

 

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar previamente se debe seleccionar el cheque que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar cheque con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el cheque que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Para habilitar el botón Depositar previamente se debe seleccionar el cheque que aún no ha sido depositado, al ser presionado se muestra la interfaz Depositar cheque que permite introducir los datos necesarios para su registro.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Comprobante previamente se debe seleccionar el cheque que desea imprimirle su comprobante, al ser presionado se muestra el comprobante de pago a FINCIMEX de dicho cheque,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada.

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.

 

  1. Al seleccionar el botón Imprimir se muestra el listado de cheques existentes,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada. En el documento en la parte superior izquierda se muestra el logo de la Entidad correspondiente.

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.

 

  1. Para consultar el saldo disponible por tipo de moneda y combustible para cada cheque, antes se debe seleccionar la unidad a la que pertenece habilitándose la opción Saldos.
