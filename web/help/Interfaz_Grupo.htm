Grupos

  •  Al seleccionar la opción Grupos de la barra de menú, se muestra la siguiente interfaz donde se gestiona los grupos que contienen los usuarios que interactúan con la aplicación como (Adicionar, Modificar y Eliminar). Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1. Al hacer clic en la opción Adicionar se muestra el siguiente formulario que permite introducir la denominación del grupo que se desea adicionar. El campo señalado indica que es obligatorio y no debe quedar vacío.

 

 

  • Para guardar los datos o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar el grupo que se desea modificar, al seleccionar la opción Modificar se muestra  la interfaz Modificar grupos. Este formulario solo modifica la denominación del grupo.

 

  • Se modifican los datos deseados,  para actualizar los cambios realizados de los datos o cancelar la operación se muestra al final de la interfaz las opciones . 

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el grupo que se desea eliminar, luego se muestra el mensaje de Confirmación.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de Confirmación las opciones  .
