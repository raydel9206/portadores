Trabajo/Proyecto

 

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Trabajo/Proyecto se muestra la interfaz Trabajos/Proyectos dividida en dos secciones, la sección #1 permite gestionar los trabajos/proyectos existentes en el sistema agrupados por Unidad definidos en dependencia del centro de costo que se le especifique,  mostrándose  los datos Código, Nombre, Fecha inicio y Fecha fin  y en la sección #2 la asignación de combustible asociado a estos trabajos/proyectos, mostrándose  los datos Tipo de combustible, Moneda y Cantidad . La interfaz muestra las opciones de gestión (Adicionar, Modificar y Eliminar) para ambos conceptos así como la opción de realizar una búsqueda introduciendo el nombre y la fecha del trabajo/proyecto que se desea encontrar, además se realiza la limpieza de la fecha mediante el botón . Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1. Para habilitar el botón Adicionar  previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así adicionar el trabajo/proyecto dando la posibilidad al usuario de especificar los datos necesarios para su registro. Al hacer clic en el mismo se muestra el siguiente formulario  que permite insertar los datos especificados por el usuario. 

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar el trabajo/proyecto que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar trabajo/proyecto con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el trabajo/proyecto que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
