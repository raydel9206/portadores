Tarjeta

  • Al seleccionar la opción del menú Economía / Tarjeta se muestra la interfaz Tarjetas listándose las tarjetas existentes en el sistema agrupadas por el centro de costo. En la interfaz se muestran las opciones que permiten gestionar el concepto tarjeta la cual se encarga de llevar el combustible que se le haya asignado así como la persona y el vehículo que la utilizan. La interfaz da la opción de dar baja a las tarjetas así como consultarlas cuando estas ya no estén en uso, además la opción de cancelar las tarjetas y consultarlas una vez canceladas, también da la posibilidad de recargar una tarjeta, darle salida y obtener un historial de todas las recargas realizadas a la misma. Estas opciones que posibilitan la gestión de las tarjetas son: Adicionar, Modificar, Cancelar, Bajas y Recargas.  La interfaz muestra la opción de realizar una búsqueda avanzada introduciendo el número de tarjeta, el tipo de combustible, el tipo de moneda y el estado en que se encuentra la tarjeta que se desea encontrar, posteriormente se ejecuta la búsqueda y se realiza la limpieza de la lista mediante los botones . En la parte derecha de la interfaz se muestra un panel que al seleccionarse una tarjeta se despliega, el mismo está dividido en 2 secciones permitiendo gestionar las personas y los vehículos que interactúan con la misma mediante las opciones Adicionar y Eliminar.  

 

 

  1. El botón Menú posee la opción de Adicionar, para habilitar la opción del menú Adicionar previamente se debe seleccionar la unidad a la que pertenece, acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro, especificando si la tarjeta es excepcional, de reserva y si tiene saldo inicial, de tener saldo inicial el campo Importe será obligatorio. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. El botón Menú posee la opción de Modificar, para habilitar el botón Modificar previamente se debe seleccionar la tarjeta que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar tarjeta con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. El botón Menú posee la opción de Canceladas, para habilitar la opción previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así cancelar la tarjeta que se desea y obtener un listado de todas las tarjetas canceladas por unidades.

 

   3.1 Para habilitar el botón Cancelar previamente se debe seleccionar la tarjeta que se desea cancelar,  al ser presionado se muestra el mensaje    de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

   3.2 Al presionar el botón Listado de Canceladas se muestra la interfaz Tarjetas Canceladas con el listado de todas las tarjetas canceladas por unidades con los datos No. tarjeta, Fecha Cancelación y mostrando el motivo de su cancelación.

 

  • Para cerrar la interfaz se muestra al final la opción

 

  1. El botón Menú posee la opción de Bajas, para habilitar la opción previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así dar baja a la tarjeta que se desea y obtener un listado de todas las tarjetas dadas de baja por unidades.

 

  4.1  Para habilitar el botón Dar Baja previamente se debe seleccionar la tarjeta que se desea dar baja,  al ser presionado se muestra el mensaje    de Confirmación que permite confirmar la operación realizada.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  4.2 Al presionar el botón Listado de Bajas se muestra la interfaz Tarjetas dadas de baja con el listado de todas las tarjetas que fueron baja por unidades con los datos No. tarjeta y Fecha de la baja.

  • Para cerrar la interfaz se muestra al final la opción

 

  1. El botón Recargas posee la opción de Acciones de Recargas, para habilitarla previamente se debe seleccionar la tarjeta con que se desea trabajar,  permitiendo así realizar diferentes acciones de recarga dependiendo del estado en que se encuentra la tarjeta.

 

  • Si el estado en que se encuentra la tarjeta seleccionada es En Tránsito, las acciones de recarga que se activan son Entrar a caja, Entrar a caja y Recargar y Entrar a caja, Recargar y Salida.

   5.1 Para habilitar el botón Entrar a caja previamente se debe seleccionar una tarjeta en estado de tránsito, al ser presionado automáticamente la tarjeta seleccionada pasa al estado En Caja.

 

  5.2 Para habilitar el botón Entrar a caja y Recargar previamente se debe seleccionar una tarjeta en estado de tránsito, al ser presionado permite definir los datos necesarios para realizar una recarga, una vez recargada la tarjeta pasa al estado Recargada en Caja. Acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  5.3 Para habilitar el botón Entrar a caja, Recargar y Salida previamente se debe seleccionar una tarjeta en estado de tránsito, al ser presionado permite definir los datos necesarios para realizar una recarga, una vez recargada la tarjeta se mantiene en el estado En Tránsito. Acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  • Si el estado en que se encuentra la tarjeta seleccionada es En Caja, las acciones de recarga que se activan son Recargar y  Salir de caja.

 5.4 Para habilitar el botón Recargar previamente se debe seleccionar una tarjeta en estado En Caja, al ser presionado permite definir los datos necesarios para realizar una recarga, una vez recargada la tarjeta pasa al estado Recargada en Caja. Acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  5.5 Para habilitar el botón Salir de caja previamente se debe seleccionar una tarjeta en estado En Caja, al ser presionado la tarjeta pasa al estado En Tránsito.

 

  • Si el estado en que se encuentra la tarjeta seleccionada es Recargada en Caja, la acción de recarga que se activa es Salir de caja.

  5.6 Para habilitar el botón Salir de caja previamente se debe seleccionar una tarjeta en estado Recargada en Caja, al ser presionado la tarjeta pasa al estado En Tránsito.

 

  1. El botón Recargas posee la opción de Historial de Recargas, para habilitarla previamente se debe seleccionar la tarjeta con que se desea trabajar, mostrando un listado de todas las recargas realizadas a la misma mediante los datos No. Factura, No.Vale, Fecha y Monto.  La interfaz muestra la opción de realizar una búsqueda introduciendo las fechas de las recargas realizadas( Desde-Hasta) que se desea encontrar, posteriormente se ejecuta la búsqueda y se realiza la limpieza de la lista mediante los botones. Además si desea realizar una corrección,  el botón Revertir permite definir los datos necesarios para ello.

  • Para cerrar la interfaz se muestra al final la opción

 

 6.1 Si se selecciona una tarjeta específica, deseando realizar una corrección, el botón Corregir permite definir los datos necesarios, para ello se especifica a partir de una fecha seleccionada y la hora en que se realiza dicha corrección. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. El botón Adicionar en la sección Vehículo permite asignar un vehículo a la tarjeta seleccionada, al hacer clic en el mismo se muestra el siguiente formulario, que permite introducir el dato necesario para su registro. El campo señalado indica que no puede quedar vacío para ser adicionado correctamente.

  • Para guardar los cambios o cancelar la operación se muestra al final de la interfaz las opciones . 

 

  1.  Si se selecciona un vehículo específico con el objetivo de eliminar la asignación, automáticamente se habilita el botón Eliminar, permitiendo eliminar el elemento seleccionado mediante el mensaje de Confirmación.

  • Para guardar los cambios o cancelar la operación se muestra al final de la interfaz las opciones . 

 

  1. El botón Adicionar en la sección Persona permite asignar una persona a la tarjeta seleccionada, al hacer clic en el mismo se muestra el siguiente formulario, que permite introducir el dato necesario para su registro. El campo señalado indica que no puede quedar vacío para ser adicionado correctamente.

  • Para guardar los cambios o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Si se selecciona una persona específica con el objetivo de eliminar la asignación realizada, automáticamente se habilita el botón Eliminar, permitiendo eliminar el elemento seleccionado mediante el mensaje de Confirmación.

  • Para guardar los cambios o cancelar la operación se muestra al final de la interfaz las opciones .
