Anticipo

  • Al seleccionar la opción del menú Economía / Anticipo se muestra la interfaz Anticipos listándose los anticipos realizados al seleccionar una Unidad. En la interfaz se muestra un Menú con las opciones que permiten gestionar el concepto anticipo de liquidaciones como Adicionar, Modificar, Cancelar y Bajas, además de la opción Cerrar, también la opción de obtención de datos Imprimir. En la interfaz se muestra la opción de realizar una búsqueda avanzada introduciendo el No. de vale, el Vehículo y el estado (Abiertos, Cerrados y Todos) del anticipo que se desea encontrar, posteriormente se ejecuta la búsqueda y se realiza la limpieza de la lista mediante los botones . En el panel denominado Liquidaciones, el cual se encuentra en la parte derecha de la interfaz se realiza la liquidación del anticipo que esté seleccionado mediante la opción Liquidar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

  1. El botón Menú posee la opción de Adicionar, esta permite registrar los datos de un anticipo mediante la interfaz Adicionar anticipo se especifican los datos Fecha, Hora para definir el tiempo en el cual se ejecuta la acción del anticipo así como su identificador mediante el campo No. Anticipo, el Vehículo, la Tarjeta, la Persona, el Trabajo/Proyecto, la Cantidad en (L), el Importe y se especifica en caso que sea necesario los campos Tránsito y Terceros.  El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para guardar los datos introducidos o cancelar la operación se muestra al final de la interfaz las opciones. 

 

  1. El botón Menú posee la opción de Modificar, para habilitar el botón Modificar previamente se debe seleccionar el anticipo que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar anticipo con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. El botón Menú posee la opción de Cancelar. Si se selecciona un anticipo específico con el objetivo de cancelarlo, automáticamente se habilita el botón Cancelar, permitiendo establecer un estado de inactividad en el elemento seleccionado, esta función se realiza mediante el mensaje de Confirmación.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de Confirmación las opciones   .

 

  1. El botón Menú posee la opción de Bajas, para habilitar la opción previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así dar baja al anticipo que se desea y obtener un listado de todas los anticipos dados de baja por unidades.

 

  4.1 Para habilitar el botón Dar Baja previamente se debe seleccionar el anticipo que se desea dar baja,  al ser presionado se muestra el mensaje    de Confirmación que permite confirmar la operación realizada.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  4.2 Al presionar el botón Listado de Bajas se muestra la interfaz Anticipos dados de baja con el listado de todas los anticipos que fueron baja por unidades con los datos No. vale y Fecha de la baja.

 

  • Para cerrar la interfaz se muestra al final la opción .

 

  1. Al seleccionar el botón Imprimir se muestra el reporte de el análisis de los consumos equipo a equipo,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada.

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.

 

  1. Si se selecciona un anticipo específico con el objetivo de cerrar sus funciones, automáticamente se habilita el botón Cerrar, se muestra la interfaz Cerrar anticipo la cual permite establecer la fecha y la hora en la cual se cierra el anticipo seleccionado. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para guardar los cambios  o cancelar la operación se muestra al final de la interfaz las opciones. 

 

  1. Si se selecciona un anticipo específico con el objetivo de realizarle una liquidación, automáticamente se habilita el panel Liquidaciones  y acto seguido se habilita el botón Liquidar, el mismo  permite realizar la liquidación del anticipo seleccionado por el siguiente formulario que permite introducir los datos necesarios para su registro mediante las opciones Adicionar y Eliminar.

  • Para guardar los cambios  o cancelar la operación se muestra al final de la interfaz las opciones.
