Familia de Productos

 

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Interfaz Familia de Productos se muestra la interfaz Familias de Productos la cual permite gestionar las familias de productos existentes en el sistema mediante las opciones Adicionar, Modificar y Eliminar. Las familias de productos existentes en el sistema se listan mediante el elemento Nombre. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación de la familia que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

  1. Para adicionar una familia de producto se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir la denominación de la familia de producto que se desea adicionar. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la familia de producto que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar familia con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la familia de producto que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
