Asignación

  • Al seleccionar la opción del menú Portadores / Combustible / Asignación se muestra la interfaz Asignaciones de Combustible listándose las asignaciones existentes en el sistema mediante los elementos Fecha, Tipo de Combustible y Cantidad. Se muestran las opciones que permiten gestionar los datos de las asignaciones como: Adicionar, Modificar y Eliminar. En la interfaz se despliega a la derecha un panel denominado Plan Disponible FINCIMEX que permite gestionar el plan disponible FINCIMEX por tipo de combustible mediante las opciones Guardar y Deshacer. En la interfaz se muestra la opción de realizar una búsqueda avanzada introduciendo la fecha y el tipo de combustible empleado que se desea encontrar, posteriormente se ejecuta la búsqueda y se realiza la limpieza del tipo de combustible mediante el botón .

 

 

  1. Para habilitar el botón Adicionar  previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así adicionar una asignación dando la posibilidad al usuario de especificar los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.    

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la asignación que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar asignación de combustible con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la asignación que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Para habilitar el botón Guardar y Deshacer previamente se debe dar doble clic en el plan disponible que desea modificar, permitiendo así modificar el plan correspondiente a cada tipo de combustible, dando la posibilidad al usuario de guardar y deshacer los datos necesarios para su registro. Al hacer clic en guardar se muestra el siguiente formulario  que permite guardar los datos especificados por el usuario.
