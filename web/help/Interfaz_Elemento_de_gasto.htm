Elemento de gasto

  • Al seleccionar la opción del menú Datos Primarios / Economía / Elemento de Gasto se muestra la interfaz Elementos de Gasto listándose los elementos de gasto definidos en el sistema mediante los elementos Código, Moneda, Combustible y Descripción. Permite gestionar el concepto elemento de gasto mediante las opciones Adicionar, Modificar  y Eliminar.

 

  1. Para adicionar un elemento de gasto se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro así como la opción de especificar si tiene combustible asignado. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar el elemento de gasto que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar Elemento de gasto con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el elemento de gasto que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
