Subsistema Portadores Energéticos

  • Para acceder al Subsistema de Portadores Energéticos debe autenticarse primero en la interfaz de inicio del sistema, en la que se debe introducir el usuario, la contraseña para acceder a él.

  • Luego de haberse registrado para tener acceso al subsistema se muestra la interfaz Seleccionar período  la cual permite seleccionar el año y el mes en el que se desea trabajar:

 

  • Luego de haber seleccionado el período que se desea trabajar se muestra la interfaz principal, que permite acceder a todas las funcionalidades que el mismo ofrece.

  • En la imagen se muestra la Interfaz Principal del Subsistema de Portadores Energéticos el cual consta de las siguientes opciones que se especifican en sus respectivos índices.

 

 

  1. El Módulo Datos Primarios agrupa todos los datos que se necesitan gestionar para lograr un correcto funcionamiento del subsistema.

  2. El Módulo Portadores agrupa toda la gestión relacionada con los Portadores Energéticos.

  3. El Módulo Transporte permite gestionar los datos referentes al  transporte, dando la posibilidad al usuario de llevar los conceptos anexo único, plan recape, prueba del litro, accidente, la paralización, el mantenimiento, hoja de ruta y reportes.

  4. El Módulo Economía permite gestionar los datos referentes a la economía como el Cheque FINCIMEX, las actas de responsabilidad material, las tarjetas así como llevar el anticipo y entrega de chips.

  5. El Módulo Reportes brinda reportes de los datos gestionados en el sistema.

  6. Muestra las notificaciones pendientes que se generen en el transcurso de la ejecución del proceso, mostrando la cantidad de las notificaciones generadas.

  7. Muestra la información del Usuario que se encuentra registrado en el sistema.

  8. Permite realizar el cambio de Subsistemas o salir de la Aplicación entre otras opciones.
