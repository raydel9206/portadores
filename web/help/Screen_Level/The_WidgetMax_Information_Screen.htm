The WidgetMax Information Screen

Overview

The WidgetMax Information screen is used to configure and track a field installed WidgetMax unit.

Fields

Field

Description

Tracking ID

This field displays the unique identification code assigned to your WidgetMax unit.

This field is display only.

Location

This field displays the current location of the WidgetMax unit. If the unit is in a location not found in our vast database, the Global Positioning Satellite (GPS) coordinates are displayed here instead.

This field is display only.

Signal Strength

This field displays the estimated signal strength of the WidgetMax unit.

This field is display only.

Alert Interval

This field displays the frequency at which the WidgetMax unit will send alerts should an unsatisfactory condition be present.

This field is display only.
