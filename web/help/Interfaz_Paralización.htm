Paralización

  • Al seleccionar la opción del menú Transporte / Paralización se muestra la interfaz Paralizaciones listándose los elementos definidos en el sistema. Se muestran las opciones que permiten gestionar la paralización de un vehículo mediante el coeficiente de disposición técnica que se muestra a la derecha de la interfaz como: Adicionar, Modificar, Eliminar y Exportar. En la interfaz se muestra la opción de realizar una búsqueda introduciendo la matrícula del vehículo que se desea encontrar. Se muestra la opción Imprimir que posibilita obtener el listado de los coeficientes de disposición técnica existentes en el sistema.

 

  1. Para adicionar una paralización antes se debe seleccionar la unidad a la que pertenece, se habilita la opción Adicionar acto seguido se muestra el siguiente formulario de información y posteriormente dando aceptar le permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar previamente se debe seleccionar la paralización que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar paralización con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la paralización que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Si se selecciona el botón Exportar, se muestra  la interfaz Exportar informe  donde le permite introducir los datos necesarios, posteriormente se muestra la opción del navegador  con la cual se obtiene el documento Actas de responsabilidad Material en formato .doc. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para obtener el documento o cancelar la operación se muestra al final de la interfaz Imprimir las opciones

 

  1. Al seleccionar el botón Imprimir se muestra el listado de los coeficientes de disposición técnica,  así como la opción del navegador ( Google Chrome) de confirmación para realizar la función señalada. En el documento en la parte superior izquierda se muestra el logo de la Entidad correspondiente.

 

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.
