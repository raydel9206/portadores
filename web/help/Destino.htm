Destino

 

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Destino se muestra la interfaz del mismo nombre la cual permite gestionar el concepto mediante las opciones Adicionar, Modificar y Eliminar. Los destinos existentes en el sistema se listan mediante el elemento Nombre. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación de el destino que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1. Para adicionar un destino se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir la denominación de el destino que se desea adicionar. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar el destino que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar destino. Este formulario solo modifica la denominación de el destino.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el destino que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
