Unidad Medida

 

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Unidad Medida se muestra la interfaz Unidades de Medida  listándose las unidades de medidas existentes en el sistema mediante el elemento Nombre. Se muestran las opciones que permiten gestionar las unidades de medida como: Adicionar, Modificar y Eliminar. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación de la unidad de medida que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

  • Para definir un modelo debe saber que: Unidad de Medida es una cantidad estandarizada de una determinada magnitud física, definida y adoptada por convención o por ley.​ Cualquier valor de una cantidad física puede expresarse como un múltiplo de la unidad de medida.

 

 

 

  1. Para adicionar una unidad de medida se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir la denominación de la unidad de medida que se desea adicionar. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

 

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la unidad de medida que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar unidad de medida. Este formulario solo modifica la denominación de la unidad de medida.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la unidad de medida que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
