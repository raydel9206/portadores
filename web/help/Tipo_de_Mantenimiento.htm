Tipo de Mantenimiento

 

  • Al seleccionar la opción del menú Datos Primarios / Transporte / Tipo de Mantenimiento se muestra la interfaz Tipos de Mantenimiento  la cual permite gestionar el concepto tipos de mantenimientos mediante las opciones Adicionar, Modificar  y Eliminar mediante los elementos Nombre y Clasificación. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación del tipo de mantenimiento que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

  1. Para adicionar un tipo de mantenimiento se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.    

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Si se selecciona un tipo de mantenimiento, deseando modificar algún dato, automáticamente se habilita el botón Modificar permitiendo modificar los datos asociados, también se habilita un panel denominado Norma por Denominación con la información de la norma de consumo para el tipo de mantenimiento seleccionado; al hacer clic sobre el mismo, se muestra el siguiente formulario mostrando los datos contenido en él, pertenecientes al elemento seleccionado. Los campos señalados indican que no puede quedar vacíos para ser adicionado el elemento correctamente.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el tipo de mantenimiento que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
