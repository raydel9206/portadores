Usuarios

  • Al seleccionar la opción del menú Usuario se muestra la interfaz del mismo nombre la cual permite gestionar los usuarios que interactúan con el sistema. El sistema muestra las opciones de gestión  Adicionar un nuevo usuario, así como Modificar, Cambiar Contraseña y Eliminar un usuario previamente seleccionado. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación del usuario que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1.  Este botón permite adicionar un nuevo usuario con los datos especificados. Al hacer clic en el mismo se muestra la siguiente interfaz que contiene las opciones,  Aceptar  y  Cancelar  las cuales se encargan de guardar los datos introducidos para un usuario o cancelar la operación realizada. Los campos señalados indican que son obligatorios y no deben quedar vacío.

 

 

 

 

  2.  Para habilitar el botón Modificar  previamente se debe seleccionar el usuario que se desea modificar, al seleccionar la opción se muestra la interfaz Modificar usuario.

 

· Se modifican los datos deseados  para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el usuario que se desea eliminar, luego se muestra el mensaje de Confirmación.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .

 

  1.  Si desea cambiar la contraseña a un usuario previamente seleccionado, debe seleccionarse la opción Cambiar contraseña mostrándose la interfaz Modificar contraseña permitiendo al usuario cambiar la contraseña existente.

 

  • Para realizar el cambio de contraseña para el usuario seleccionado o cancelar la operación se muestra al final de la interfaz las opciones .
