Vehículo

  1. Al seleccionar la opción del menú Datos Primarios / Transporte / Vehículo se muestra la interfaz Vehículos listándose los vehículos existentes en el sistema mediante los elementos Matrícula, Unidad, Tipo de combustible, Norma, Norma FAR, No. inventario, No. circulación, Fecha expiración circulación y No. Orden. Se muestran las opciones que permiten gestionar los datos de los vehículos como: Adicionar, Modificar y Eliminar. Al seleccionar un vehículo se despliega a la derecha de la interfaz un panel denominado Otros Datos que permite gestionar los Mantenimientos y los Choferes del vehículo seleccionado mediante las opciones Adicionar, Modificar y Eliminar. En la interfaz se muestra la opción de realizar una búsqueda avanzada introduciendo la matrícula del vehículo y el tipo de combustible empleado por este que se desea encontrar, posteriormente se ejecuta la búsqueda y se realiza la limpieza del tipo de combustible mediante el botón . Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

  1. Para habilitar el botón Adicionar  previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así adicionar un vehículo dando la posibilidad al usuario de especificar los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.    

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar el vehículo que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar vehículo con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el vehículo que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
