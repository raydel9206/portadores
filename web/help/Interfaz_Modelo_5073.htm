Modelo 5073

  • Al seleccionar la opción del menú Reportes / Combustible / Modelo 5073 se muestra la interfaz del mismo nombre mostrando los datos en la plantilla obtenida de los resultados del proceso de gestión del Sistema Portadores dando la posibilidad al usuario de introducir los datos necesarios para su registro. La interfaz muestra  las opciones que permiten generar el Modelo 5073 y guardar los cambios mediante las opciones: Generar Modelo y Guardar Cambios, así como obtener el registro de los datos mediante las opciones Imprimir y Exportar, también se muestran las opciones de paginado ya que la plantilla puede exceder la cantidad máxima de elementos permitidos por páginas, así como la barra de desplazamiento.

 

  1. Para habilitar la opción Generar Modelo previamente se debe seleccionar la unidad a la que pertenece, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Para habilitar la opción Guardar Cambios previamente se debe dar doble clic en los datos, dando la posibilidad al usuario de actualizar los mismos para su registro, posteriormente al hacer clic en el botón Guardar Cambios  se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Al seleccionar el botón Imprimir se muestra el reporte de el Modelo 5073,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada.

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.

 

  1. Al seleccionar el botón Exportar  se muestra la opción del navegador que permite obtener la información en un documento excel (.xls).
