Portador

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Portador se muestra la interfaz Portadores listándose los portadores existentes en el sistema mediante los elementos Nombre y Unidad de Medida. Se muestran las opciones que permiten gestionar los portadores  como: Adicionar, Modificar y Eliminar. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación de el portador que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1. Para adicionar un portador se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir la denominación de el portador que se desea adicionar así como la unidad de medida. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar  el portador que se desea modificar, al seleccionar la opción se muestra la interfaz Modificar portador con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el portador que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
