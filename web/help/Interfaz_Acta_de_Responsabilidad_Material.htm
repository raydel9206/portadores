Acta de Responsabilidad Material

 

  • Al seleccionar la opción del menú Economía / Acta de Responsabilidad Material se muestra la interfaz del mismo nombre listándose las actas de responsabilidad material definidas en el sistema. En la interfaz se muestran las opciones que permiten  definir  y obtener un acta de responsabilidad material con las opciones: Adicionar, Modificar, Eliminar, Imprimir  y Exportar.  La interfaz muestra la opción de realizar una búsqueda avanzada introduciendo el número de tarjeta, posteriormente se ejecuta la búsqueda y se realiza la limpieza de la lista mediante los botones .

 

 

  1. Para adicionar un acta de responsabilidad material antes se debe seleccionar la unidad a la que pertenece, se habilita la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar previamente se debe seleccionar el acta de responsabilidad material que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar acta con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

 

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el acta de responsabilidad material que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

 

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. Al seleccionar el botón Imprimir se muestra el listado de cheques existentes,  así como la opción del navegador (Google Chrome) de confirmación para realizar la función señalada.

 

  • Para obtener la impresión del documento o cancelar la operación se muestra las opciones Imprimir o Cancelar.

 

  1. Al seleccionar el botón Exportar  se muestra la opción del navegador que permite obtener la información en un documento excel (.xls).
