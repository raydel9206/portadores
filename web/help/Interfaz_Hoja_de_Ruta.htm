Hoja de Ruta

  • Al seleccionar la opción del menú Transporte / Hoja de Ruta se muestra la interfaz Hojas de Ruta listándose los elementos definidos en el sistema mediante los datos No. Hoja, Matrícula y Fecha. Permite gestionar el concepto hoja de ruta mediante las opciones Adicionar, Modificar  y Eliminar. La interfaz permite realizar la creación de una hoja de ruta para un vehículo en específico, al ser seleccionado una hoja de ruta se activa el panel frontal que se encarga de gestionar las rutas y los conductores que se establezcan en la hoja de ruta, permitiendo gestionar mediante las opciones Adicionar y Eliminar.  En la interfaz también se  muestra la opción de realizar una búsqueda seleccionando la matrícula del vehículo. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

  1. Para adicionar una hoja de ruta antes se debe seleccionar la unidad a la que pertenece, se habilita la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir los datos necesarios para su registro. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar previamente se debe seleccionar la hoja de ruta que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar hoja de ruta con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la hoja de ruta que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. El botón Adicionar  referente a Rutas permite adicionar una o varias  rutas a una hoja de ruta previamente seleccionada con los datos necesarios para su registro, al hacer clic en el mismo se muestra el siguiente formulario, que permite introducir los datos. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar una ruta de una hoja de ruta, eliminando el elemento seleccionado mediante el mensaje de Confirmación.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones

 

  1. El botón Adicionar  referente a Conductores permite adicionar varios conductores a una hoja de ruta previamente seleccionada con los datos necesarios para su registro, al hacer clic en el mismo se muestra el siguiente formulario, que permite introducir los datos. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar un conductor, eliminando el elemento seleccionado mediante el mensaje de Confirmación.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones
