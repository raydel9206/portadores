Pie de Firma

 

  • Al seleccionar la opción del menú Datos Primarios / Estructura / Pie de Firma se muestra la interfaz Pie de Firmas de Documentos  listándose los pies de firmas editados en el sistema para los documentos mediante las opciones Documento, Confeccionado por, Revisado por, Aprobado por  y Cajera.  Se mostraran los elementos definidos en dependencia del nivel estructural que se le especifique a la Unidad. Se muestran las opciones que permiten gestionar los pies de firma  como: Adicionar, Modificar y Eliminar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1. Para habilitar el botón Adicionar  previamente se debe seleccionar la unidad a la que pertenece, permitiendo así adicionar un pie de firma  de un documento determinado, dando la posibilidad al usuario de especificar los datos necesarios para su registro. Al hacer clic en el mismo se muestra el siguiente formulario  que permite insertar los datos especificados por el usuario.  

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar el pie de firma que se desea modificar, al seleccionar la opción se muestra la interfaz Modificar Pie de Firma con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

  • Debe tener en cuenta que si modifica la estructura de un Pie de Firma esto requerirá cambios en el sistema y su funcionamiento.

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar el pie de firma que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .

  • Debe tener en cuenta que si elimina la estructura de un Pie de Firma esto requerirá cambios en el sistema y su funcionamiento.
