Subsistema Administración

  • Para acceder al Subsistema de Administración debe autenticarse primero en la interfaz de inicio del sistema, en la que se debe introducir el usuario, la contraseña para acceder a él.

  • Luego de haberse registrado para tener acceso al subsistema se muestra la interfaz Seleccionar período  la cual permite seleccionar el año y el mes en el que se desea trabajar:

  • Luego de haber seleccionado el período que se desea trabajar se muestra la interfaz principal, que permite acceder a todas las funcionalidades que el mismo ofrece.

  • En la imagen se muestra la Interfaz Principal del Subsistema Administración el cual consta de las siguientes opciones que se especifican en sus respectivos índices.

 

  1. El Módulo Grupos se encarga de gestionar los grupos donde se ubicarán los usuarios, para asignar los permisos de las  funcionalidades de forma asociativa.

  2. El Módulo Usuarios permite gestionar los usuarios que existen en el sistema con los datos básicos y necesarios para su registro.

  3. El Módulo Recursos permite otorgarle a los grupos los permisos necesarios para el acceso a las funcionalidades del sistema.

  4. El Módulo Notificaciones permite gestionar las notificaciones que se generen en el sistema.

  5. El Módulo Eventos permite mediante una búsqueda avanzada encontrar  las funciones realizadas en dependencia de los datos especificados.

  6. Muestra las notificaciones pendientes que se generen en el transcurso de la ejecución del proceso, mostrando la cantidad de las notificaciones generadas.

  7. Muestra la información del Usuario que se encuentra registrado en el sistema.

  8. Permite realizar el cambio de Subsistemas o salir de la Aplicación entre otras opciones.
