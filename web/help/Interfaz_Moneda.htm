Moneda

  • Al seleccionar la opción Datos Primarios / Economía / Moneda, se muestra la interfaz denominada Tipos de Moneda listando los elementos definidos por el dato Tipo de moneda. La interfaz muestra las opciones de gestión Adicionar, Modificar y Eliminar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

  • Para definir una moneda debe saber que: se define como tipo de moneda a cualquier tipo de dinero que está en circulación en una economía. comúnmente, la palabra moneda hace referencia a cualquier forma de dinero utilizado por todo el público y que se encuentra en circulación.

 

  1. Para adicionar una moneda se debe presionar la opción Adicionar acto seguido se muestra el siguiente formulario que permite introducir la moneda que se desea adicionar. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la moneda que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar moneda con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la moneda que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
