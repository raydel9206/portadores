Actividad

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Actividad se muestra la interfaz Actividades la cual permite gestionar las actividades agrupadas por los portadores del sistema mediante las opciones Adicionar, Modificar y Eliminar. Las actividades existentes en el sistema se listan mediante los elementos Nombre, Nombre UM Actividad, Código GAE, Código MEP, Actividad  no Productiva, Inversión y Tráfico. La interfaz muestra la opción de realizar una búsqueda introduciendo la denominación de la actividad que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

  1.  El botón Adicionar permite adicionar una Actividad dando la posibilidad al usuario de especificar los datos necesarios para su registro como Nombre, Nombre UM Actividad, Código GAE, Código MEP, Actividad  no Productiva, Inversión y Tráfico. Al seleccionar la opción se muestra la interfaz Adicionar actividad  que permite insertar los datos especificados por el usuario. El campo señalado indica que es obligatorio y no debe quedar vacío para su correcto registro.   

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar  la actividad que se desea modificar, al seleccionar la opción se muestra la interfaz Modificar actividad con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones .

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la actividad que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
