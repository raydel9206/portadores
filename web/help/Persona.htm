Persona

 

  • Al seleccionar la opción del menú Datos Primarios / Gestión / Persona se muestra la interfaz Personas la cual permite gestionar las personas agrupadas por las unidades mediante las opciones Adicionar, Modificar  y Eliminar. Se mostraran los elementos definidos en dependencia del nivel estructural que se le especifique a la Unidad,  mostrándose  los datos Nombre(s) y Apellidos, CI, Dirección y Teléfono. La interfaz muestra la opción de realizar una búsqueda introduciendo el nombre de la persona que se desea encontrar. Al final de la interfaz se muestran las opciones de paginado dado que la cantidad de elementos existentes puede exceder  la cantidad de elementos permitidos por páginas.

 

 

 

  1. Para habilitar el botón Adicionar  previamente se debe seleccionar la unidad a la que pertenece,  permitiendo así adicionar la persona dando la posibilidad al usuario de especificar los datos necesarios para su registro así como el Cargo y Operario de taller. Al hacer clic en el mismo se muestra el siguiente formulario  que permite insertar los datos especificados por el usuario. 

  • Para registrar los datos introducidos  o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Modificar  previamente se debe seleccionar la persona que se desea modificar, al seleccionar el formulario Modificar se muestra  la interfaz Modificar persona con los datos contenidos en el elemento seleccionado, dando la posibilidad de modificarlos siguiendo las especificaciones del campo.

  • Para actualizar los cambios realizados a los datos modificados o cancelar la operación se muestra al final de la interfaz las opciones

 

  1. Para habilitar el botón Eliminar  previamente se debe seleccionar la persona que se desea eliminar, al ser presionado se muestra el mensaje de Confirmación que permite confirmar la operación realizada.

  • Para confirmar o cancelar la operación se muestra al final del mensaje de confirmación las opciones .
